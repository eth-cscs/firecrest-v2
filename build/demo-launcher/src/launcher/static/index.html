Examples

The following are a set of curl command examples you can copy and paste in your terminal to quicky interact with the FirecREST API.

Submit Job:
curl -X POST --header "Content-Type: application/json" --header "Authorization: Bearer " -d '{ "job": {"name": "Count to 100", "working_directory": "", "script": "#!/bin/bash\nfor i in {1..100}\ndo\necho $i\nsleep 1\ndone"}}' "http://localhost:5000/compute//jobs"
List Jobs:
curl --header "Authorization: Bearer " "http://localhost:5000/compute//jobs"
List folder content:
curl --header "Authorization: Bearer " "http://localhost:5000/filesystem//ops/ls?path="
